# E-commerce Sales Analysis Dashboard

![Screenshot 2024-03-04 002626](https://github.com/AdityaPatil100/Power-Bi-projects/assets/86911300/3860ee67-5c34-4520-92dd-901c34b3cacb)

A comprehensive Power BI dashboard designed to analyze e-commerce sales performance across various dimensions including geography, time, and product categories.

## Project Overview

This project visualizes key performance indicators (KPIs) and sales trends for an e-commerce business. The dashboard provides actionable insights into profitability, customer behavior, and regional sales distribution to help drive data-driven decision-making.

## Key Features

* **KPI Tracking:** Real-time visibility of Sum of Amount (438K), Total Profit (37K), Quantity Sold (5615), and Average Order Value (121K).
* **Monthly Profit Analysis:** A bar chart showing profit fluctuations throughout the year, highlighting seasonal trends and loss-making months (e.g., May and July).
* **Regional Performance:** Breakdown of sales by state, identifying Maharashtra and Madhya Pradesh as top-performing regions.
* **Product Insights:** * **Category Analysis:** Sales distribution across Clothing, Electronics, and Furniture.
* **Sub-Category Profitability:** Detailed view of profit margins for specific items like Printers, Bookcases, and Accessories.


* **Customer Analysis:** Identification of top customers by total spend (e.g., Harivansh, Madhav).
* **Interactive Filters:** Slicers for Quarter (Q1-Q4) and State to drill down into specific data segments.

## Tech Stack

* **Power BI Desktop:** Used for data transformation (Power Query), modeling, and visualization.
* **Dataset:** E-commerce sales records including transaction details, customer names, and geographic data.

## Visualizations Included

1. **Donut Charts:** Sum of Quantity by Payment Mode and Category.
2. **Bar & Column Charts:** Profit by Month, Amount by State, and Profit by Sub-Category.
3. **Cards:** High-level summary of total sales metrics.
4. **Slicers:** Time-based and regional filters for interactive reporting.

## How to Use

1. Clone the repository.
2. Open the `.pbix` file using **Power BI Desktop**.
3. Interact with the filters on the top right to explore specific quarters or states.
